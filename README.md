A minimal, keyboard-first To-Do web app built with plain HTML/CSS/JS.
Features:
- Inline edit (double click / press `E`)
- Keyboard shortcuts: `Enter` to add, `E` to edit, `Del` to delete focused, `Ctrl+S` to export JSON
- Filters: All / Active / Completed
- LocalStorage persistence
- Import / Export JSON for portability
- Responsive, accessible, and developer-oriented styling

## Install / Run
1. Clone this repo
2. Open `index.html` in the browser or run a simple static server (e.g. `npx http-server`)
